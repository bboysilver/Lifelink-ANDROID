# LifeLink 출시 로드맵

기준 버전: Android 2.2.2 (23)

## 현재 목표: Closed testing

### 제출 전 확인

- 앱 ID는 `com.bboysilver.lifelink`를 유지한다.
- GitHub Actions의 `Signed Android release`에서 생성한 서명 AAB만 사용한다.
- 최종 AAB Manifest에 `com.google.android.gms.permission.AD_ID`가 없어야 한다.
- Play Console의 광고 ID 및 광고 선언은 모두 `No`로 유지한다.
- 기존 Closed testing 및 Internal testing의 오래된 아티팩트는 2.2.2 (23)으로 교체하거나 비활성화한다.
- SMS 권한 용도는 고령자 안전을 위한 물리적 안전·긴급 알림으로 설명한다.
- Foreground service는 실제 Manifest와 동일하게 `health`만 선언한다.
- 개인정보처리방침 URL은 `https://bboysilver.github.io/Lifelink-ANDROID/privacy-policy.html`을 사용한다.

### 테스트 운영

- Play Console에 표시된 조건에 따라 최소 12명의 테스터가 14일 연속 참여하도록 한다.
- Android 9~16, 삼성·픽셀·보급형 단말, 단일 SIM·멀티 SIM을 포함한다.
- 최초 설정, 테스트 SMS, 재부팅 복구, 절전 모드, SIM 제거, 비행기 모드, 권한 거부를 확인한다.
- 무활동 경보, 매일 안부 확인, SOS를 각각 반복 시험한다.
- 문자 누락, 중복 발송, 잘못된 정상 표시, 조용한 서비스 중단을 출시 차단 결함으로 처리한다.

### Closed testing 통과 기준

- 테스트 SMS 발송 콜백 성공률 99% 이상
- 동일 사건의 중복 긴급 SMS 0건
- 서비스 heartbeat 중단 상태의 정상 표시 0건
- 재부팅 후 모니터링 복구 성공
- 앱 기여 배터리 사용량 하루 5% 이하
- 위치를 수집하거나 임의 좌표를 전송하지 않음

## 1단계: Android 무료 버전 공개

예상 기간: Closed testing 완료 후 2~4주

- Closed testing 결함을 수정하고 회귀 테스트를 통과한다.
- 스토어 설명, 스크린샷, Data Safety, SMS 권한, Health apps, FGS 선언을 실제 코드와 일치시킨다.
- 핵심 안전 기능은 무료로 유지한다: 로컬 무활동 감지, 보호자 1~3명 SMS, 매일 안부 확인, SOS.
- 공개 출시는 보조 안전 도구이며 119·의료기기·전문 보안 서비스를 대체하지 않는다고 명확히 고지한다.

## 2단계: LifeLink Family MVP

예상 기간: Android 무료 버전 공개 후 8~12주

- 보호받는 사용자와 보호자 역할을 같은 앱에 추가한다.
- 초대 코드 또는 QR로 보호자를 연결한다.
- 서버 heartbeat로 휴대전화 전원 꺼짐, 네트워크 단절, 앱 중단을 추정한다.
- 보호자에게 정상·주의·확인 필요 상태와 마지막 연결 시각만 제공한다.
- 보호자 푸시, SMS, 다음 보호자 순서의 단계별 대응을 구현한다.
- 원시 센서 데이터와 상시 위치는 서버에 저장하지 않는다.

## 3단계: 구독 출시

예상 기간: Family MVP 실증 후 4~6주

- 무료와 `LifeLink Family` 한 가지 유료 등급만 운영한다.
- Google Play Billing을 사용하고 구매 토큰은 백엔드에서 검증한다.
- 갱신, 해지, 유예, 결제 보류, 환불과 구매 승인을 서버에서 처리한다.
- 구독이 끝나도 무료 로컬 감지와 긴급 SMS는 계속 동작하게 한다.
- 앱 내부와 웹에서 계정 및 서버 데이터 삭제 기능을 제공한다.
- 결제·계정·서버 데이터 처리가 추가되기 전에 개인정보처리방침과 Data Safety를 갱신한다.

## 4단계: iOS 및 보호자 웹

예상 기간: Family 구독 안정화 후 12~16주

- iOS는 백그라운드 자동 SMS가 불가능하므로 서버 푸시와 서버 발송 알림을 중심으로 별도 설계한다.
- 보호자 웹에서는 가족 상태, 마지막 연결, 경보 이력과 확인 중 상태만 제공한다.
- Apple 구독은 StoreKit과 서버 영수증 검증을 사용한다.
- Android와 iOS가 같은 보호자 연결 및 heartbeat 백엔드를 공유하게 한다.

## 공개 출시 Go/No-Go

다음 항목 중 하나라도 충족하지 못하면 공개 출시를 보류한다.

- 30가구 이상이 4주간 사용한 실기기 결과가 없음
- 긴급 SMS 누락 또는 중복 발송이 재현됨
- 앱이 멈췄는데 정상 모니터링으로 표시됨
- 정책 선언과 실제 권한·데이터 처리가 다름
- 고객 문의 및 안전 사고 대응 절차가 없음
