LifeLink 개인정보처리방침

시행일: 2026년 7월 21일

LifeLink Android 2.2.0은 위치 정보, Firebase, Twilio, 광고 SDK, 결제 SDK 또는 외부 클라우드를 사용하지 않습니다.

1. 처리하는 정보와 목적

  • 기기 별칭, 보호자 이름·전화번호: 긴급 문자의 수신자와 사용자를 식별하기 위해 사용합니다.
  • 모니터링 설정, 최근 활동 시각, 서비스 상태: 무활동 시간을 계산하고 모니터링 상태를 표시하기 위해 사용합니다.
  • 매일 안부 확인 시각·응답일, SOS 요청 시각: 안부 미응답과 사용자의 직접 도움 요청을 보호자에게 알리기 위해 사용합니다.
  • 선택한 SMS SIM과 활성 SIM 상태: 문자를 보낼 이동통신 회선을 확인하고 선택하기 위해 사용합니다.
  • 문자 발송·전달 결과와 동작 기록: 실패 여부와 재시도 상태를 사용자에게 보여주기 위해 사용합니다. 문자 상태 기록은 90일 후 정리됩니다.
  • 배터리 잔량: 긴급 문자에 기기 상태를 포함하기 위해 발송 시점에 확인합니다.

2. 저장과 전송

위 정보는 사용자의 Android 기기 내부에 저장되며 Android 백업 대상에서 제외됩니다. 무활동, 안부 미응답 또는 SOS 상황에는 기기 별칭, 요청 시각과 배터리 상태가 사용자가 직접 등록한 보호자에게 이동통신사의 SMS로 전송됩니다. 개발자는 해당 문자 내용이나 전화번호를 서버로 수집하지 않습니다.

3. 권한

  • 문자(SEND_SMS): 등록된 보호자에게 테스트 문자와 긴급 문자를 보냅니다.
  • 전화 상태(READ_PHONE_STATE): SMS 지원 여부와 활성 SIM을 확인하고, 멀티 SIM 중 사용자가 선택한 회선으로 문자를 보내기 위해 사용합니다. 전화 통화 내용이나 기기 식별번호는 수집하지 않습니다.
  • 신체 활동(ACTIVITY_RECOGNITION): 휴대전화 움직임을 활동 신호로 감지합니다.
  • 알림(POST_NOTIFICATIONS): 모니터링 실행 상태와 사전 경고를 표시합니다.
  • 부팅 완료(RECEIVE_BOOT_COMPLETED): 재부팅 후 사용자가 켜 둔 모니터링을 복구합니다.

4. 보유 기간과 삭제

연락처와 설정은 사용자가 앱 안에서 삭제하거나 Android 설정에서 앱 데이터를 삭제할 때까지 보관됩니다. 문자 발송 상태 기록은 최대 90일 보관한 뒤 자동 정리되며, 앱의 기록 삭제 기능을 사용하면 즉시 함께 삭제됩니다. 앱을 삭제하면 기기에 저장된 앱 데이터도 삭제됩니다.

5. 제3자 제공과 외부 처리

개발자는 정보를 별도 서버에 수집하거나 제3자에게 판매·제공하지 않습니다. 사용자가 등록한 보호자에게 보내는 SMS는 사용자의 이동통신사 문자 서비스를 거칩니다.

6. 안전 기능의 한계

LifeLink는 119, 의료기기 또는 전문 보안 서비스를 대신하지 않습니다. 전원 꺼짐, 앱 강제 종료, 제조사 절전 정책, SIM·통신망 장애 시 감지나 문자 발송이 동작하지 않을 수 있습니다.
